2.Tahun Kabisat

Pseudocode

STORE "Tahun" with any value

IF "Tahun" mod 4 equal 0 and "year" mod 100 not equal 0 THEN
  DISPLAY "Tahun Kabisat"
ELSE IF "Tahun" mod 4 equal 0 and "Tahun" mod 100 equal 0 and mod 400 equal 0 THEN
  DISPLAY "Tahun Kabisat"
ELSE
  DISPLAY "bukan Tahun Kabisat"
END IF

Natural

SIMPAN "tahun" dengan nilai 0

JIKA "tahun" modulus 4 sama dengan 0 dan "tahun" modulus 100 tidak sama dengan 0 MAKA
  TAMPILKAN "tahun Kabisat"

  TAMPILKAN "bukan tahun Kabisat"

4.Periksa Kuku

Pseudocode

STORE "siswa/siwi" with 1

WHILE "siswa/siswi" less than 40 THEN
  DISPLAY "siswa/siswi"
  STORE "panjang kuku" with any value
  IF "panjang kuku" greater than 4 THEN
    DISPLAY "hukum!"
  ELSE
    DISPLAY "muji"
